# pref-shimane-cms

pref-shimane-cms バージョン2は以下の機能を提供します。

## 機能一覧
* フォルダ管理機能
* ページ管理機能
* 所属、ユーザ管理機能
* フォルダ・ページ移動機能
* フォルダコピー機能
* ページ複製機能
* イベントカレンダー機能
* ブログ機能
* リンク切れチェック機能
* 広告管理機能
* 相談窓口検索機能
* 音声ファイル作成機能
* 一括ページ取り込み機能
* アクセシビリティチェック機能(JIS X 8341-3:2010 AA基準)
